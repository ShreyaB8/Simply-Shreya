Projects

Hardware & Software

Below are a list of some of my interesting projects. All of them are on GitHub, but due to reasons related to academic integrity, most of them are private.

The Firefly Unit

Electronics, Communication Network

A mobile ad-hoc network communication system using Zigbee and LoRaWAN protocols for IoT applications in Remote Locations. This project provides a reliant communication system for people to have a long-distance transmission of information.
Worked with Manet, Zigbee, Data routing and clustering algorithm

Webin-AR

Web Dev, Augmented Reality, Speech synthesis

An all-in-one Education platform with integrated interactive AR models, 3D models, Video chat classroom, and speech synthesis capability.

Worked in the speech to text, text to speech, speech processing, backend processing

Predictive Smart Machine Authorizer

Electronics, PCB fabrication, Web Dev, ML

This project gives control over the access to heavy machinery along with monitoring the health of every machine. All the data is concisely presented on the front end and the logs in the database.
My work involved: Circuit designing, PCB designing, Backend, Microcontroller

MedX

Web Dev, Blockchain

MedX is a Web App made using Etherium Blockchain to track and maintain Corona test records in a manageable and reliable way.

Worked with Backend using Node.js, Vanilla Js, and Web3 APIs

DASH: Direct Automated Shelf Helper

freeRTOS, Electronics, PCB Fabrication, AWS

DASH automatically creates a shopping list for your groceries using the smart IoT sensor that is inbuilt in it. Developed the product from scratch till deployment.
My work involved: freeRTOS, Battery optimization, and power management, Circuit designing, PCB designing, AWS architecture

RCMN: Road Condition Monitoring Network

Web Dev, IoT

RCMN is an end-to-end solution for monitoring the condition of roads and bring critical failures to the notice of the responsible personnel. The system automatically passes the raised ticket to higher authorities to ensure that the condition is maintained.
Worked with Sensor Packages and Backend
